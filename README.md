## Custom Action (Node.js)

### Requirements
- Node.js 12.22.1

### Setup
```cmd
git clone https://github.com/ivanyu199012/10.-KofaxRPA-Tips---Custom-Action
cd 10.-KofaxRPA-Tips---Custom-Action\NodeJs\node_modules\sampleHandler

npm install
cd ../../

node test.js
```
